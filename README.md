# ai-workshop-assignment
Assignment for the AI workshop
The repo contains a SKILL and reference context file. You can use following prompt to investigate self-reported income reliability:
“Using skill.md and reference.md, give me an overview of the reliability of the income reliability - how much does it under/overstate and where - in the UK portfolio”

In addition, you will be prompted to confirm assumptions and add potential extra segmentations you might be interested in.
